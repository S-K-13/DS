# Step 2: Import SparkSession
from pyspark.sql import SparkSession

# Step 3: Create Spark Session
spark = SparkSession.builder.appName("WordCount").getOrCreate()

# Step 4: Sample Data (can also use file)
data = ["Hello Spark Hello Hadoop", 
        "Spark is fast", 
        "Hadoop and Spark"]

# Convert data to RDD
rdd = spark.sparkContext.parallelize(data)

# Step 5: Apply Word Count Logic

# Split words
words = rdd.flatMap(lambda line: line.split(" "))

# Map each word to (word, 1)
word_pairs = words.map(lambda word: (word, 1))

# Reduce to get count
word_counts = word_pairs.reduceByKey(lambda a, b: a + b)

# Step 6: Display Output
print("Word Count Result:")
for word, count in word_counts.collect():
    print(word, count)

# EXPERIMENT 13 – APACHE SPARK (WORD COUNT USING PYSPARK)

## AIM
To perform a simple data processing task using Apache Spark by implementing a Word Count program.

## CODE EXPLANATION

### 1. INSTALLING PYSPARK
!pip install pyspark  
Installs PySpark library in Colab environment.  
Why: Spark is not pre-installed in Colab.

### 2. IMPORTING SPARK SESSION
from pyspark.sql import SparkSession  
SparkSession → entry point to Spark functionality.  
Why: Required to initialize Spark.

### 3. CREATING SPARK SESSION
spark = SparkSession.builder.appName("WordCount").getOrCreate()  
builder → configures Spark  
appName() → sets application name  
getOrCreate() → creates new session or uses existing  
Why: Starts Spark engine.

### 4. INPUT DATA
data = ["Hello Spark Hello Hadoop","Spark is fast","Hadoop and Spark"]  
List of strings used as sample dataset.  
Why: Input for processing.

### 5. CREATING RDD
rdd = spark.sparkContext.parallelize(data)  
parallelize() → converts data into RDD (Resilient Distributed Dataset).  
Why: Spark processes data using RDDs.

### 6. SPLITTING WORDS (flatMap)
words = rdd.flatMap(lambda line: line.split(" "))  
flatMap() → splits each line into words  
Why: Converts sentences into individual words.

### 7. MAPPING WORDS
word_pairs = words.map(lambda word: (word, 1))  
map() → creates key-value pairs (word,1)  
Why: Prepares data for counting.

### 8. REDUCING (COUNTING)
word_counts = word_pairs.reduceByKey(lambda a, b: a + b)  
reduceByKey() → groups same words and adds counts  
Why: Calculates frequency of each word.

### 9. DISPLAY OUTPUT
for word, count in word_counts.collect(): print(word, count)  
collect() → brings results to driver  
Why: Displays final result.

## WORKING FLOW
Input data → convert to RDD → split words → map (word,1) → reduceByKey → display output

## KEY CONCEPTS
RDD → distributed data structure  
flatMap → transformation (split data)  
map → transformation (key-value pair)  
reduceByKey → aggregation  
collect → action (retrieve result)

# -------------------------------------------------
# Assignment 13: Apache Spark Word Count (PySpark)
# -------------------------------------------------

from pyspark.sql import SparkSession

# Create Spark Session
spark = SparkSession.builder.appName("WordCount").getOrCreate()

# Sample Data
data = ["Hello Spark Hello Hadoop",
        "Spark is fast",
        "Hadoop and Spark"]

# Convert to RDD
rdd = spark.sparkContext.parallelize(data)

# Word Count Logic
words = rdd.flatMap(lambda line: line.split(" "))
word_pairs = words.map(lambda word: (word, 1))
word_counts = word_pairs.reduceByKey(lambda a, b: a + b)

# Display Output
